
  # INCH BY INCH

  This is a code bundle for INCH BY INCH. The original project is available at https://www.figma.com/design/DKMwlT6gs1XxxkUPsxplSf/INCH-BY-INCH.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploying to Vercel (auto-deploy on push)

  This repo includes a GitHub Actions workflow that deploys to Vercel on every push to `main`.

  1. Import the repo into Vercel: [lUCIFER94008/Inchbyinch](https://github.com/lUCIFER94008/Inchbyinch.git)
  2. In your GitHub repo, add these **Repository secrets**:
     - VERCEL_TOKEN
     - VERCEL_ORG_ID
     - VERCEL_PROJECT_ID
  3. Push to `main` (or run the workflow manually from the Actions tab).
